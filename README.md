# Setup

Require node.js 10.x higher.(Recommended 12.x)

```
npm install -g yarn && yarn
```

# Develop

```
yarn dev
```

# CSS Rule

ぶっちゃけると決めてないけどレイアウト系には独自のクラス名を付けてフォントの大きさとか色とか見た目は専用のクラスを作ってそれを個別に当てる運用にしたいなって思ってる（適当）
